The problems with the iPhone timer can be resolved in a variety of ways. Here is one of the solutions that will resolve the timer's interface problem. This interface allows you to set the time simply by clicking the desired buttons, eliminating the need for a scroll wheel.

Hours


Minues


As the prototype shown above shows. Instead of using minutes and seconds like a microwave, my updated iPhone timer uses hours and minutes. The user has the option of choosing from a number of buttons to set the timer for. Therefore, you would choose buttons 1 and 0 for the hours and then 2 and 7 for the minutes if you wanted a 10 hour and 27 minute timer. The use of this design would enhance the user experience and enhance the original design as a whole.
